# Cooper Young Portfolio

Personal portfolio showcasing software development, AI projects, and Concord.

## Sections

- About
- Concord AI Intelligence Platform
- Technical Skills
- Experience
- Certifications
- Contact

## Tech Stack

- Next.js
- TypeScript
- Tailwind CSS

## Setup

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

## Build

```bash
npm run build
```
